All the Tools You Need

Every utility has been carefully crafted with real-time processing, intuitive interfaces, and seamless copy-paste functionality.

Timestamp Converter

Convert between Unix timestamps and human-readable dates with timezone support and multiple format options.

Unit Converter

Convert between different units across 7 categories: Data, Time, Length, Weight, Temperature, Area, and Volume.

JSON Formatter

Format, validate, and compare JSON data with visual CodeMirror diff editor and syntax highlighting.

SQL Formatter

Format and minify SQL queries with syntax validation and proper indentation for better readability.

HTML Formatter

Format and minify HTML with proper tag indentation and structure validation.

Base64 Encode/Decode

Encode and decode Base64 strings with support for URL-safe variants and automatic detection.

JWT Encoder/Decoder

Encode and decode JSON Web Tokens with HMAC and RSA signature support using CryptoKit security.

Regex Test

Test regular expressions with pattern matching, capture groups, and common pattern templates.

UUID Generator

Generate UUIDs in various formats (v1, v4, v5, v7) with bulk generation and timestamp extraction.

URL Tools

URL encoding/decoding and comprehensive URL parsing with component breakdown and validation.

HTTP Request

Full-featured HTTP client with method selection, headers, auth, response handling, and JSON tree view.

IP Query

Discover your IP address and query geolocation data for any IP with detailed location information.

QR Code

Generate QR codes with multiple sizes and error correction levels, scan QR codes from images.

Parquet Viewer

Read and explore Parquet/Arrow files with schema inspection, data export, and unified Rust-based API.

Crypto Tools

Comprehensive cryptographic suite with hash functions (MD5, CRC32, SHA family), symmetric encryption (AES), and asymmetric encryption (RSA).
